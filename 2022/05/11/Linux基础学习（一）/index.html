Linux基础学习（一）

第一部分 Linux的规则与安装

第5章 首次登录与在线求助man page

命令

  • X windows快速重启：Alt+Ctrl+Backspace
  • 文字登录界面切换：Ctrl+Alt+F1~F6
  • 图形登录界面切换：Ctrl+Alt+F7
  • 启动X窗口：startx
  • 注销Linux：exit
  • 修改语言：LANG=en_us
  • 显示日期与时间：date
  • 显示日历：cal
  • 计算器：bc
  • 停止：Ctrl+c
  • 结束输入：Ctrl+d
  • 在线求助：man 命令、info 命令
  • 查看系统的使用状态：
    （1）查看目前在线用户：who
    （2）查看网络联机状态：netstat -a
    （3）查看后台执行程序：ps -aux
  • 数据同步写入磁盘：sync
  • 关机命令：
    （1）shutdown [-t 秒] [-arkhncfF] 时间 [警告信息]
    （2）halt 可以执行硬件关机的特殊功能
    （3）reboot
    （4）poweroff
  • 切换执行等级：init
    （1）run level 0：关机
    （2）run level 3：纯命令行模式
    （3）run level 5：含有图形界面模式
    （4）run level 6：重启

    第二部分 Linux、目录与磁盘格式

    第6章 Linux的文件权限与目录配置

    6.2.1 Linux文件属性

    文件的类型与权限

    （1）d 目录
    （2）- 文件
    （3）l 连接文件
    （4）b 可供存储的接口设备
    （5）c 串行端口设备
    （6）r 可读
    （7）w 可写
    （8）x 可执行

    命令

  • 列出所有文件详细的权限与属性：ls -al
  • 改变所属用户组：chgrp [-R] dirname/filename
  • 改变文件所有者：chown
    （1）chown [-R] 账号名称 文件或目录
    （2）chown [-R] 账号名称：组名 文件或目录
  • 改变权限：chmod
    （1）三种身份：owner、group、others
    （2）9个基本权限：-rwxrwxrwx
    （3）权限分数：r-4、w-2、x-1
    （4）chmod [-R] xyz 文件或目录
    （5）chomd [u/g/o/a] +/-/= r/w/x 文件或目录

    6.2.4 文件种类与拓展名

    文件种类及属性符号

  • 普通文件：-
    （1）纯文本文件
    （2）二进制文件
    （3）数据格式文件
  • 目录：d
  • 连接文件：l
  • 设备与设备文件
    （1）块设备文件：存储数据以提供系统随机访问的接口设备 b
    （2）字符设备文件：串行端口的接口设备 c
  • 套接字：s
  • 管道FIFO：p

    Linux文件拓展名

  • *.sh：脚本或批处理文件
  • *Z、*.tar、*.tar.gz、*.zip、*.tgz：压缩文件
  • .html、.php：网页相关文件

    6.3 Linux目录配置

    6.3.1 Linux目录配置标准：FHS

  • 四种交互作用的形态
    （1）可分享的
    （2）不可分项的
    （3）不变的
    （4）可变动的
  • 三层目录
    （1）/：与开机系统有关
    （2）/usr：与软件安装/执行有关，放置的数据属于可分享的与不可变动的，“UNIX操作系统软件资源”放置目录
    （3）/var：与系统运作过程有关，主要针对常态性变动的文件，包括缓存、登录文件以及某些软件运行所产生的文件
  • / 下子目录（FHS标准规定）
    （1）/bin：放置在单用户维护模式下还能够被操作的命令，不可与根目录放在不同分区
    （2）/boot：放置开机会使用到的文件
    （3）/dev：放置文件形式的设备与接口设备，不可与根目录放在不同分区
    （4）/etc：放置主要的配置文件，不可与根目录放在不同分区
    （5）/home：默认的用户主文件夹
    （6）/lib：放置开机会用到的函数库以及/bin或/sbin下命令会调用的函数库，不可与根目录放在不同分区
    （7）/media：放置可删除的设备
    （8）/mnt：用来暂时挂载某些额外的设备
    （9）/opt：第三方软件放置的目录
    （10）/root：系统管理员的主文件夹
    （11）/sbin：放置开机过程中需要的，包括开机、修复、还原系统的命令，不可与根目录放在不同分区
    （12）/srv：放置一些网络服务启动后需要取用的数据
    （13）/tmp：放置一般用户或正在执行的程序暂时放置文件的地方
  • / 下子目录（其他重要）
    （1）/lost+found：放置一些在文件系统发生错误时丢失的片段
    （2）/proc：一个虚拟文件系统，放置的数据在内存当中，不占硬盘空间
    （3）/sys：一个虚拟文件系统，记录与内核相关的信息，不占硬盘空间
  • /usr下子目录
    （1）/usr/X11R6/:放置X Window系统重要数据
    （2）/usr/bin/：放置绝大部分的用户可使用命令
    （3）/usr/include/：放置C/C++等程序语言的头文件与包含文件
    （4）/usr/lib/：放置各应用软件的函数库、目标文件以及不被一般用户惯用的执行文件或脚本
    （5）/usr/local/：放置自行下载安装的软件
    （6）/usr/sbin/：放置非系统正常运行所需要的系统命令
    （7）/usr/share/：放置共享文件的地方
    （8）/usr/src/：放置源码
    • /var下子目录
      （1）/var/cache/：放置应用程序运行过程中会产生的一些暂存文件
      （2）/var/lib/：放置在程序执行过程中需要使用到的数据文件
      （3）/var/lock/：设备锁
      （4）/var/log/：放置登录文件
      （5）/var/mail/:放置个人电子邮件信箱的目录
      （6）/var/run/：放置某些程序或者服务启动后的PID
      （7）/var/spool/：放置一些队列数据

      6.3.3 绝对路径与相对路径

  • . :代表当前的目录，也可以使用 ./ 来表示
  • .. :代表上一层目录，也可以使用 ../ 来表示

    6.3.4 Centos的查看

  • 查看实际的内核版本：uname -r
  • 查看distribution的具体版本信息：lsb_release -a

    第7章 Linux文件与目录管理

    7.1 目录与路径

    7.1.1 相对路径与绝对路径

  • 绝对路径：路径的写法一定由根目录/写起
  • 相对路径：路径的写法不是由/写起，指相对于目前工作目录的路径

    7.1.2 目录的相关操作

  • 比较特殊的目录：
    （1）. :代表此层目录
    （2）.. ：代表上一层目录
    （3）- ：代表前一个工作目录
    （4）~ ：代表“目前用户身份”所在的主文件夹
    （5）~account ：代表account这个用户的主文件夹
  • 常见处理目录的命令
    （1）切换目录：cd [相对路径或绝对路径]
    （2）显示当前目录：pwd [-p]
    -p 显示出当前的路径，而非使用连接（link）路径
    （3）新建一个新的目录：mkdir [-mp] 目录名称
    -m配置目录的权限（强制）
    -p 递归创建需要的目录
    （4）删除一个空的目录：rmdir [-p] 目录名称
    -p 递归删除空目录
    （5）删除目录及目录下所有东西：rm -r 目录名称

    7.2 文件与目录管理

    7.2.1 查看文件与目录： ls

  • ls [-aAdfFhilnrRSt] 目录名称
    （1）-a ：全部的文件，连同隐藏文件一起列出来
    （2）-d ：仅列出目录本身
    （3）-l ：列出长数据串，包含文件的属性与权限等数据
  • ls [--color={never,auto,always}] 目录名称
  • ls [--full-time] 目录名称

    7.2.2 复制、删除与移动：cp,rm,mv

  • cp [-adfilprsu] 源文件 目标文件
    （1）-a ：相当于-pdr
    （2）-i ：若目标文件已经存在时，在覆盖时会先询问操作的进行
    （3）-p ：连同文件的属性一起复制过去，而非使用默认属性
    （3）-r ：递归持续复制，用于目录的复制行为
  • cp [options] source1 source2 source3 ... directory
  • rm [-fir] 文件或目录
    （1）-f :忽略不存在的文件，不会出现警告信息
    （2）-i ：互动模式，在删除前会询问用户是否操作
    （3）-r ：递归删除
  • mv [-fiu] source destination
    （1）-f :强制
    （2）-i ：若目标文件已经存在，就会询问是否覆盖
    （3）-u ：若目标文件已经存在，且source比较新，才会更新
  • mv [options] source1 source2 source3 directory

    7.2.3 取得路径的文件名与目录名称

  • basename 路径 ：取得文件名
  • dirname 路径 ：取得目录名

    7.3 文件内容查阅

    7.3.1 直接查看文件内容

  • cat [-AbEnTv] 文件路径 ：由第一行开始显示文件内容
    （1）-A ：相当于 -vET的整合参数，可列出一些特殊字符
    （2）-b ：列出行号，仅针对非空白行做行号显示
    （3）-n ：打印出行号，连同空白行也会有行号
  • tac 文件路径 ：由最后一行开始显示
  • nl [-bnw] 文件 ：显示的时候输出行号

    7.3.2 可翻页查看

  • more 文件路径 ：一页一页地显示文件内容
  • less 文件路径 ：一页一页地显示文件内容（可以往前翻页）

    7.3.3 数据选取

  • head [-n number] 文件 ：取出前面几行
  • tail [-n number] 文件 ：取出后面几行

    7.3.4 非纯文本文件：od

  • od [-t TYPE] 文件
    （1）a 利用默认字符来输出
    （2）c 利用ASCII字符来输出
    （3）d 利用十进制来输出数据
    （4）f 利用浮点数来输出数据
    （5）o 利用八进制来输出数据
    （6）x 利用十六进制来输出数据

    7.3.5 修改文件时间或创建新文件：touch

  • 三个时间参数
    （1）mtime ：文件内容数据最后更新时间（ls默认显示的时间）
    （2）ctime ：文件状态最后更新时间
    （3）atime ：文件内容最后读取时间
  • ls -l --time=时间参数 文件
  • touch [-acdmt] 文件
    （1）-d：后接欲修改的日期
    （2）-t：后接欲修改的时间，格式[YYMMDDhhmm]
